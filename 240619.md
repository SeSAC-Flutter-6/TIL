### 정규식

- 패턴을 체크하는 것
모든 언어에서 다 지원함
- r: row(원시) - r이 있으면 글자로 인식하지 않고 없으면 반대
- .. 패턴: 두 글자 달라도 match 됨
- Dart에서 문자열을 정규 표현식으로 나누려면 `RegExp` 객체를 사용하여 `split` 메서드를 호출할 수 있다

```dart
String text = '1,2,3';
RegExp pattern = RegExp(r',');
List<String> parts = text.split(pattern);
print(parts); // 출력: ['1', '2', '3']
```
