- subString(start,end)
  * subString(start,end) 는 문자열에서 start Index 부터
    end Index까지를 잘라서 String 객체로 리턴합니다
  * 주의: start Index는 문자열에 포함되고 end Index는 포함되지 않습니다
  * 즉, (end - 1) Index까지 잘라진 문자열에 포함됩니다
